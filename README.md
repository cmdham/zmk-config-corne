# Corne Keyboard Config
## Information
My ZMK config for a Corne wireless keyboard. For use with macOS, yabai, raycast etc.

### yabai/skhd 
- LWR+1-4, move to space
- ⇧LWR+1-4, move window to space
- ⌥+hjkl, move focus
- ⌥LWR+hjkl, move window

### other
- ⌘+space, raycast
- ^⌘+space, raycast emoji
- fn, toggle betwabeen EN/JP
- ⌘+tab, alt tab
